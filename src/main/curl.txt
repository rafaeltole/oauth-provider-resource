--- obter token
curl -X POST -k --user 'resource:secret' -d 'grant_type=client_credentials' http://localhost:8180/oauth-provider-authorization/oauth/token

--- check token
curl -X POST -k --user 'resource:secret' -d 'grant_type=client_credentials' -d "token=3047cf86-9453-4fe1-a7b2-12c2b241c5e8" http://localhost:8180/oauth-provider-authorization/oauth/check_token
